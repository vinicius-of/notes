Requisitos:
- Spring Boot 3 e Java 22
- Spring Web MVC
- Spring Data JPA
- Spring Validation
- Spring Hateoas

Endpoints:
- POST - /products
- GET - /products
- GET - /products/{id}
- PUT - /products/{id}
- DELETE - /products/{id}

Para gerar o projeto: [start.spring.io](start.spring.io).
